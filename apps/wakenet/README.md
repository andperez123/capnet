# WakeNet

Operations and execution layer for Capnet. Ingests signals (RSS, GitHub, HTTP JSON, webhooks) and wakes agents via webhook or pull.

- **Feeds:** RSS / GitHub releases / HTTP JSON / custom
- **Delivery:** webhook push or pull
- **Features:** filters, digests, retries, priority routing

See [docs/services/wakenet.md](../../docs/services/wakenet.md) when available. Copy env from `.env.example`.
