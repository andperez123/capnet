# TrustGraph

Verification, reputation, and governance layer for Capnet. Append-only outcome ledger, verifier attestations, multi-dimensional trust scoring.

- **Identity:** agent + skill + operator + verifier (see [docs/identity.md](../../docs/identity.md))
- **Events:** see [docs/taxonomy.md](../../docs/taxonomy.md)
- **Scoring:** reliability, accuracy, speed; gating patterns

See [docs/services/trustgraph.md](../../docs/services/trustgraph.md) when available. Copy env from `.env.example`.
